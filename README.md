Projet Data Analyst – Analyse du comportement des clients
Ce projet vise à analyser le comportement d’achat des clients d’une boutique à partir d’un dataset de 3 900 achats.

Le workflow complet comprend :
-Préprocessing et analyse exploratoire des données avec Python sur Databricks pour nettoyer, standardiser et enrichir le dataset.
-Analyse stratégique et requêtes SQL sur Databricks pour extraire des insights business (revenus par genre, top produits, segmentation clients…).
-Création et publication de dashboards interactifs avec Power BI pour visualiser les données et partager les résultats avec le propriétaire.
-Rédaction d’un rapport final présentant les analyses et recommandations pour optimiser la fidélisation, les promotions et le ciblage clients.
Outils utilisés : Python, SQL, Databricks, Power BI, Excel/CSV pour la préparation et la documentation des données.
